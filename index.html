PocketWiKi

PocketWiKi is a lightweight android application to interact with WiKipedia without having to open the website explicitly and search for the content. It is a simple and easy to use application that allows users to search for content on WiKipedia and view it in the application itself. It also allows users to save their favourite articles for later use. The application is built using Java and Android Studio. It uses the WiKipedia API to fetch the content and display it in the application.

Link: PocketWiKi

Not-so-Simple Quizzard!

Not-so-Simple Quizzard is a super simple quiz platform developed with a 'security-first' approach.

The main highlights of the project are:

  • > PHP based backend with MySQL database
  • > No user authentication
  • > Unique Identifier tracking for scores
  • > No session tracking
  • > No cookies
  • > Data Duplication Prevention
